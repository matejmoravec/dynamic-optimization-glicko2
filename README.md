# [ARTICLE] A Novel Performance Indicator for Dynamic Optimization Based on Rating Systems

This repository provides a reference implementation and experimental framework for evaluating dynamic optimization algorithms
(DOAs) using an extension of the CRS4EAs (Chess Rating System for Evolutionary Algorithms) method. The proposed methodology
enables performance comparison without requiring knowledge of the global optimum, making it particularly suitable for real-world
dynamic optimization problems.

## Usage

### Dependencies

This project depends on the **EARS (Evolutionary Algorithm Rating System)** framework, which can be obtained from the
official repository: [https://github.com/UM-LPM/EARS](https://github.com/UM-LPM/EARS)

### Project Structure

- **`results/cec2025`**  
  Contains the official results from the **IEEE CEC 2025 Competition on Dynamic Optimization Problems Generated by the
  Generalized Moving Peaks Benchmark (GMPB)**.
  
  *Although there are currently no DOAs or DOPs implemented in the EARS framework, users can still utilize this framework
  for comparison and ranking purposes using the CRS4EAs method. The user can use any other existing framework to execute
  DOAs on selected DOPs. The framework can be implemented in any programming language, or the user can implement both the
  DOA and DOP manually. The user must only provide the results in the proper format that can be read by the EARS framework.
  The results for the selected DOAs on the selected DOPs must be provided in the CEC format, where each column represents
  a specific run of the DOA and each row represents the result (fitness value) at a specific evaluation which depends on
  the set cutpoint interval when using the proposed method to draw the rating confidence graph. More details about how to
  format the results in the CEC format can be found in
  [*CEC2024 Comparison Slides.pdf*](https://github.com/P-N-Suganthan/2024-CEC/blob/main/CEC2024%20Comparison%20Slides.pdf).

- **`results/edolab`**  
  Contains experimental results obtained using the **EDOLAB platform**:  
  https://github.com/EvoMindLab/EDOLAB

- **`src/main/kotlin/benchmark/cec2025`**  
  Contains benchmark implementations:
  - **`GMPBCEC2025.kt`**  
    Benchmark for evaluating algorithms across **all dynamic optimization problems (DOPs)** or **groups of DOPs** defined
    by specific configuration settings.
  - **`GMPBCutpoint.kt`**  
    Benchmark designed to generate **rating confidence bands**.
  - **`GMPBSingle.kt`**  
    Benchmark for evaluating algorithms on a **single dynamic optimization problem**.
